# IEC 60870 5-104 Import Script Template

## Repository Goal

This repository provides a template for creating automation scripts that integrate with the IEC 60870-5-104 connector using its Import Script feature. The template offers a structured, connector-compatible format that enables developers to retrieve external data without altering the connector itself.

## Projects Overview

- [SLC-AS-IEC_60870_5-104_ImportScriptTemplate](SLC-AS-IEC_60870_5-104_ImportScriptTemplate/README.md) - An automation script template designed for use with the Import Script feature of the IEC 60870-5-104 connector. It provides a pre-validated framework for seamless integration with the connector.
