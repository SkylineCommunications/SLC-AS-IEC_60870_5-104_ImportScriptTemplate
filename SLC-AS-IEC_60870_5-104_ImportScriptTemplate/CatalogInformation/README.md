# IEC 60870 5-104 Import Script

## About

This template provides a starting point for creating automation scripts that integrate with the [IEC 60870-5-104 connector](https://catalog.dataminer.services/details/003758df-a239-49a7-9679-73b6023a1d9e) using its Import Script feature.

## Key Features

The template provides a structured, connector-compatible format with essential placeholders, ensuring seamless integration and compliance with expected standards. It enables you to:

 - Retrieve external data effortlessly without altering the connector, maintaining stability and compatibility.
 - Accelerate development by leveraging a pre-validated script framework, reducing setup time and minimizing errors.
